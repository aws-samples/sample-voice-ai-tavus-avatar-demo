AWS Voice AI Overview

Real-time multimodal voice agents on AWS.

This proof-of-concept page is designed for the voice AI tool-call demo. It is same-origin, safe to embed in an iframe, and concise enough to display while the avatar keeps talking in the corner overlay.

Speech Pipeline

Daily transports real-time audio and video, while Tavus renders the avatar and receives visual context from the camera feed.

Reasoning Layer

Amazon Bedrock handles language understanding and response generation with tool calls available for screen control.

Production Pattern

The same flow can move from a private booth demo to a local Electron app with minimal UI changes.

Demo Stack

Amazon SageMaker AI Amazon Bedrock Daily WebRTC Tavus CVI Deepgram speech models Pipecat orchestration
